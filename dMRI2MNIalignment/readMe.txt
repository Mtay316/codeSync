-----------------
Procedures
-----------------
1. Follow warp.txt or run warp_batch.sh
2. Apply intensity_normalization.py to each subject so that their intensity are comparable
3. If files too large to be concatenated into 1 group, rename the files with dwi*_1.nii to separately concatenate each group
4. Follow concat.txt
5. If using trackVis or older version of TTK, apply reorder_B0_top.py in dMRIanalysis folder
